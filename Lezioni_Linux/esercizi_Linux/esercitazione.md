# Esercitazione

1. Crea le seguente struttura nella cartella documenti
- progetto
    - docs
    - backup
    - test
    - main

In docs crea i seguenti file: README.md e CHANGELOG.md

In test crea i seguenti file: test1.js e test2.js

In main crea i seguenti file: main.js e utils.js

2. Crea una copia dei soli file presenti nelle singole cartelle all'interno della cartella backup

3. Cambia nome a tutti i file della cartella backup che hai copiato precedentemente aggiungendo BU prima dell'estensione (es: main.js -> mainBU.js)

4. verifica con il comando tree tutta la struttura creata. Al termine lancia il comando history e copia, a mano, tutti i comandi utilizzati. Incolla questi comandi in un foglio di testo che allegherai al task di classroom
